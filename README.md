# Krimex Portfolio

Персональный портфолио-сайт в стиле neo-brutalism с информацией о проектах, услугах и разработках.

## Особенности

- Минималистичный дизайн в стиле neo-brutalism
- Адаптивная вёрстка для всех устройств
- Лёгкий и быстрый встроенный Python-сервер
- Простой деплой без зависимостей

## Установка и запуск

**Требования:** Python 3.7+

```bash
git clone https://github.com/Krimex1/krimex-portfolio.git
cd krimex-portfolio
python app.py
```

Сайт будет доступен по адресу `http://localhost:8000`

Для изменения порта:
```bash
PORT=3000 python app.py
```

## Структура

# Krimex Portfolio

Персональный сайт-портфолио в стиле neo-brutalism.

## Запуск

```bash
git clone https://github.com/Krimex1/krimex-portfolio.git
cd krimex-portfolio
python app.py
```

Сайт доступен по адресу `http://localhost:8000`
